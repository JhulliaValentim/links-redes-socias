# 🔗 Link in Bio | Portfólio Profissional

![Status do Projeto](https://img.shields.io/badge/Status-Concluído-success?style=for-the-badge)
![Vercel](https://img.shields.io/badge/Deploy-Vercel-black?style=for-the-badge&logo=vercel)

Uma página web responsiva, elegante e centralizada no formato "Link in Bio", desenvolvida para agrupar o meu portfólio profissional, redes sociais e formas de contacto num único lugar.

**Acesso ao  projeto:** [jhullialinksredes.vercel.app](https://jhullialinksredes.vercel.app/)

---

## Visualização (Dark Theme)
O design foi pensado com um tema escuro moderno, focando no contraste suave com tons de roxo (`#2a2231`) e destaques em rosa envelhecido (`#c98a8e`), garantindo uma leitura confortável e um aspeto profissional.

## Tecnologias Utilizadas
Este projeto foi construído puramente com as tecnologias base da web, garantindo o máximo de velocidade e leveza:
* **HTML5** (Semântica e estruturação)
* **CSS** (Estilização, Flexbox, Variáveis CSS para o tema)
